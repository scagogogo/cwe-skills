# 单元测试覆盖率提升到 100% 计划

> **For agentic workers:** REQUIRED SUB-SKILL: `superpowers:subagent-driven-development`
> Steps use checkbox (`- [ ]`) syntax.

**Goal:** 将整个 cwe-skills 项目的 Go 单元测试覆盖率从当前 71.2% 提升到 100%（SDK 层从 97.4% 到 100%，cmd/cwe 与 cmd/cwe-mcp 两层从 0% 到 100%），并在 CI 中加覆盖率门禁。

**Architecture:** 当前覆盖率：SDK 根层 97.4%（18 个函数低于 100%）、cmd/cwe 0%（17 个可测函数无测试）、cmd/cwe-mcp 0%（11 个可测函数无测试）、总 71.2%。数据流：按层补测试——先 SDK 层（纯函数+已有测试扩充），再 cmd/cwe 辅助纯函数（printJSON/parseIDArg/boolStr 等），再 cmd/cwe cobra 命令 RunE（构造 cobra.Command 捕获 stdout），再 cmd/cwe-mcp 辅助函数（wrapJSON/requireStringArg/toInt/serializeTreeNode），再 cmd/cwe-mcp register 函数（注册后断言工具数），最后 CI 加 -coverprofile 门禁。关键组件：现有测试风格（table-driven，见 cwe_utils_test.go）、cobra Command 测试模式（cmd.RunE + cmd.OutOrStdout buffer）、mcp server 构造（server.NewMCPServer）。为什么这样做：纯加测试不改源码，按依赖层级从底到顶，每层独立验证覆盖率提升。

**Tech Stack:** Go 1.25, testing 标准库, spf13/cobra v1.10.2, mark3labs/mcp-go v0.20.0, go tool cover, GitHub Actions

**Risks:**
- 100% 是激进目标，`main()` 函数和 `init()` 不可直接测 → 缓解：main() 不计入（Go cover 自动跳过空函数体或用 build tag），init() 通过其副作用（命令注册到 rootCmd）间接测，聚焦所有业务逻辑函数达 100%
- `Execute()` 调用 os.Exit 退出 → 缓解：测试 Execute 时用不触发退出的子命令，或重构 Execute 使其可测（若需改源码则单独标注）
- `loadRegistry`/`loadNavRegistry` 依赖 XML 文件 → 缓解：用项目内 testdata 或 cweskills 包的测试 Registry 构造
- cmd/cwe-mcp register 函数需真实 MCPServer → 缓解：`server.NewMCPServer("test","1.0")` 构造，注册后 `s.Tools()` 断言数量
- 在线 API 函数（GetWeakness 等）94% 缺最后几行（错误分支）→ 缓解：用 httptest.Server 模拟错误响应
- 某些函数的不可达 error 分支 → 缓解：若确认不可达，加注释说明并用 //go:cover 或接受 99.x%，不强求 100% 到硬写假测试

func TestParseIDArg(t *testing.T) {

		input   string
		want    int

		{"plain number", "79", 79, false},
		{"CWE format", "CWE-79", 79, false},
		{"empty", "", 0, true},
		{"invalid", "abc", 0, true},
		{"zero", "0", 0, true},
		{"negative", "-1", 0, true},
	}

	for _, tt := range tests {
		t.Run(tt.name, func(t *testing.T) {
			got, err := parseIDArg(tt.input)
			if (err != nil) != tt.wantErr {
				t.Errorf("parseIDArg(%q) error=%v wantErr=%v", tt.input, err, tt.wantErr)

				t.Errorf("parseIDArg(%q) = %d, want %d", tt.input, got, tt.want)
			}
		})
	}
}

package main

import "testing"

func TestExecuteVersion(t *testing.T) {
	// Execute 内部调 rootCmd.Execute()，version 子命令不触发 os.Exit
	// 通过设置 os.Args 模拟
	oldArgs := os.Args
	defer func() { os.Args = oldArgs }()
	os.Args = []string{"cwe", "version"}
	// Execute 调用 os.Exit 仅在 rootCmd.RunE 错误时；version 正常返回
	// 用 recovered 捕获避免测试进程退出
	defer func() { _ = recover() }()
	Execute()
}
